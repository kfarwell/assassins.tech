Assassins.TECH

Assassins.TECH is a twist on the classic game Assassin using SMS text messaging.

Text NEW to +1 (647) 797-KILL to start a new game. Text HELP for help.

Go to Assassins.TECH/<6-digit game code>.html to check a game's leaderboards.

Assassins.TECH was built in 24 hours for THacks 2 by Kyle Farwell and Jared Kelly. Read more on Devpost.
